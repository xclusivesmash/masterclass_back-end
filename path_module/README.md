# Path module

## Description
> The path module provides a way to work with paths and directories.

### Use cases
- File system interaction
- Path manipulation
- Platform independence
- Security (path travel attacks)
- Code readability and maintanability

## Requirements
- OS: `Ubuntu 24.04 LTS`
- language: `node.js`
- Editor: `vim`
